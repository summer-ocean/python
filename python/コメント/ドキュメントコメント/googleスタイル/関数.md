# 関数
```python
"""説明文

（詳細な説明）

Args:
	x (型): 引数の説明
	y: 引数の説明 （型を記述しない場合）

Returns:
	型: 戻り値の説明

Raises:
	例外obj: 例外の説明
"""
```

## 引数
```python
def func():
	"""
	Args:
	x (型): 引数の説明
	y: 引数の説明 （型を記述しない場合）
	"""
```

## 戻り値
```python
def func():
	"""
	Returns:
	型: 戻り値の説明
	"""
```

## 例外
```python
def func():
	"""
	Raises:
	例外obj: 例外の説明
	"""
```

## 使用例
```python
def func():
	"""
```
